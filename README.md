# Teste Node.js + Vue.js

## Instructions to run

1. Setup database and write connections to file backend/app/config/db.config.js

2. Run backend server with command: (inside dir backend)

```node server.js```

4. Run frontend with command: (inside dir frontend)

```npm run serve```
